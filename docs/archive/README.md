# Archive

This directory contains historical snapshots and superseded planning documents. They are kept for
traceability, not as current operating instructions.

Current guidance lives in the root [`../../README.md`](../../README.md). Paths inside archived
documents may reflect the repository layout at the time they were written.

Archive contents:

- `blueprints/` - original docx blueprint/playbook inputs.
- `experiment_changelog.md` - dated experiment history.
- `project_control_legacy.md` - former oversized project-control README snapshot.
- `removed_guides/` - markdown guides replaced by the canonical root README.
